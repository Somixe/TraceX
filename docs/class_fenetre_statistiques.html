Affiche un histogramme animé représentant les taux de traçabilité.

Cette méthode principale orchestre l'affichage du graphique dans un widget Qt donné. Elle initialise les données, prépare la scène graphique, trace les axes, ajoute les barres animées et insère le graphique dans le conteneur.

Parameters
conteneurLe widget Qt (QWidget*) dans lequel insérer l'histogramme.
listeTauxListe des taux de traçabilité (entre 0 et 100 %).
